# LibDecoration
用于RecyclerView的分割线类库

##引用

```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
```
	dependencies {
	        implementation 'com.github.q1113225201:ItemDecoration:1.0.0'
	}
```
